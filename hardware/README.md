DG-100 Enclosure Hardware Files
================================

Files in this directory:

SMOKE CHAMBER (print in matte black PETG):
  DataGuard_Chamber_Base.stl  — Bottom half with optics
  DataGuard_Chamber_Lid.stl   — Top half with air inlets
  smoke_chamber.scad          — OpenSCAD source (parametric)

ELECTRONICS ENCLOSURE (print in white/grey PETG):
  DG100_Enclosure_Base.stl    — Base with standoffs, vents, cable holes
  DG100_Enclosure_Lid.stl     — Lid with chamber mount ring, branding pad
  DG100_Enclosure_Assembly.stl — Combined (preview only)

PRINT SETTINGS:
  Chamber: Matte black PETG, 0.2mm, 20% infill, supports ON
  Enclosure: White/grey PETG, 0.2mm, 25% infill, supports ON

DIMENSIONS:
  Enclosure: 160 x 110 x 55mm
  Chamber: 65mm dia x 45mm
  Total assembled: 160 x 110 x 104mm

POST-PRINT:
  1. Ream LED/PD bores with 5mm drill
  2. Ream cable gland holes with 12mm drill
  3. Insert PG7 cable glands (back face)
  4. Mount PCB on M2.5 standoffs
  5. Screw lid with 4x M2.5x12mm screws
  6. Snap smoke chamber onto lid mount ring
  7. Apply branding label on raised pad
